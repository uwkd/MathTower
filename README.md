# Math Tower - Playable Ad Demo

Đây là dự án **Playable Ad** được phát triển nhằm mục đích ứng tuyển vị trí Playable Ads Developer.

## 🎮 Demo & Preview
- **Link chơi thử:** https://uwkd.itch.io/math-tower-playable-ad-portfolio
- **Thể loại:** Puzzle / Hyper-casual.
- **Nền tảng:** Web Mobile (HTML5).

## 🛠 Công nghệ sử dụng
- **Engine:** Cocos Creator 3.x
- **Ngôn ngữ:** TypeScript
- **Kỹ thuật:** - Xử lý va chạm dựa trên logic toán học.
  - Hiệu ứng Tween cho nhân vật và UI.
  - Tối ưu dung lượng file (dưới 10MB) để chạy mượt trên trình duyệt di động.

## 🚀 Tính năng chính
- Logic tính điểm cộng dồn khi tiêu diệt quái vật.
- Hệ thống bẫy tâm lý (Psychological Trap) đặc trưng của Playable Ads để kích thích hành vi tải game.
- Nút Call-to-Action (CTA) với hiệu ứng thu hút sự chú ý.
